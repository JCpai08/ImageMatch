# 快速开始 (Quick Start)

## 编译项目

1. 确保已安装必要的依赖：
   - 如果使用系统 OpenCV：需要安装完整 OpenCV 开发包
   - 如果使用本地精简版：确保 lib 目录下有 OpenCV 库文件
2. 创建构建目录并进入：
    ```bash
   mkdir build
   cd build
3. 配置编译：
    ```bash
   cmake ..
   make
4. 编译完成后，可执行文件位于 build/bin/Image_Match：
    ```bash
    ./bin/Image_Match
# 实验结果对比
